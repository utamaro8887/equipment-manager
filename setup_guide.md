# スプレッドシート & GAS 連携環境セットアップガイド

GASのコードをローカル環境からコマンドツール（`clasp`）で管理・デプロイするために、以下の手順で環境を準備してください。

## 1. Node.js のインストール
システムの確認を行ったところ、現在 `npm`（Node.jsのパッケージ管理ツール）がインストールされていないようです。
`clasp` を使用するには、Node.js が必要です。

1. [Node.js 公式サイト (https://nodejs.org/ja/)](https://nodejs.org/ja/) にアクセスし、**推奨版 (LTS)** をダウンロードしてインストールします。
2. インストールが完了したら、PCを再起動するか、使用しているターミナル（VS Code等）を再起動してください。

## 2. スプレッドシートの新規作成
1. Googleドライブを開き、「新規作成」＞「Google スプレッドシート」をクリックします。
2. スプレッドシート名は「備品管理データベース（仮）」などとしてください。
3. スプレッドシートのURLから「スプレッドシートID」をコピーしておいてください。
   - URLの例: `https://docs.google.com/spreadsheets/d/【ココがIDです】/edit`

## 3. clasp のセットアップとログイン
Node.js インストール後、ターミナルで本プロジェクトのフォルダ（`C:\Users\tokim\Desktop\Project\設備QR管理`）を開き、以下のコマンドを順番に実行してください。

```bash
# プロジェクトの初期化と clasp のインストール
npm init -y
npm install -D @google/clasp @types/google-apps-script

# Googleアカウントへのログイン
npx clasp login
```
※ `clasp login` を実行するとブラウザが開き、Googleの認証画面が表示されます。対象のGoogleアカウントでログインし、アクセスを「許可」してください。

## 4. GASプロジェクト（Webアプリ）の作成
連携するGASプロジェクトを作成し、現在準備している `gas` フォルダに紐づけます。
同じターミナルで以下を実行してください。

```bash
npx clasp create --type standalone --rootDir gas
```
このコマンドが成功すると、`gas` フォルダ内に設定ファイルが生成されます。
