# スプレッドシート列構成定義書

## T_Assets シート (Sheet Name: T_Assets)
備品の基本情報を管理するマスターテーブルです。

| 列 | フィールド名 | プログラム用キー | 説明 |
| :--- | :--- | :--- | :--- |
| A | ID | `id` | システム内部で利用する一意の識別子（行番号-1） |
| B | 型番 | `model_number` | 製品の型番・品番 |
| C | 備品管理番号 | `asset_tag` | カテゴリ＋年月で自動採番 (例: 車椅子2603) |
| D | カテゴリ名 | `category_id` | 備品の種類（車椅子、ベッド等） |
| E | 備品名称 | `name` | 個別の備品名称 |
| F | 現在の状態 | `status` | 稼働中\|故障中\|予備\|廃棄待\|廃棄済 |
| G | 管理部署名 | `dept_id` | 管理を担当している部署 |
| H | 設置階 | `floor` | 物理的な保管場所（例: 3F） |
| I | 設置場所 | `location` | 物理的な保管場所（例: ナースステーション） |
| J | QRアクセスキー | `qr_token` | **Unique Token** (ログイン不要アクセス用) |
| K | 説明書リンク | `manual_url` | 取扱説明書(PDF)等のURL |
| L | 購入年月日 | `purchase_date` | 備品を導入した日 |
| M | 購入業者 | `vendor` | 備品を導入した業者 |
| N | 購入金額 | `price` | 備品を導入した金額 |
| O | 耐用年数 | `useful_life` | 備品の耐用年数 |
| P | 修理依頼先 | `repair_vendor` | 備品の修理依頼先 |
| Q | 備考 | `note` | 備考 |

## T_History シート (Sheet Name: T_History)
点検や修理、移動などのイベントを記録する履歴テーブルです。

| 列 | フィールド名 | プログラム用キー | 説明 |
| :--- | :--- | :--- | :--- |
| A | 報告日時 | `timestamp` | メンテナンスや移動が報告された日時 |
| B | 対象備品ID | `asset_id` | T_Assets.id への参照キー |
| C | 報告種別 | `type` | 修理\|移動\|点検\|棚卸\|廃棄 |
| D | 担当者名 | `operator` | 報告を行った職員名 (自己申告) |
| E | 備考・メモ | `note` | 自由記述（「：なし」はこの項目の内容を表示） |

## マスタシート (M_Depts / M_Categories)
将来的な正規化用のマスタです。現在はIDまたは直接名称を保持します。
