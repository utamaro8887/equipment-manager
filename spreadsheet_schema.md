# スプレッドシート列構成定義書

スプレッドシートの「備品マスタ」シートの構成をここに定義します。
列を追加・変更した場合は、該当する列番号（A, B, C...）を書き換えてください。

## 備品マスタ シート (Sheet Name: 備品マスタ)

| 列番号 | フィールド名 | プログラム上の識別名 | 説明 |
| :--- | :--- | :--- | :--- |
| A | 備品ID | `id` | ユニークなID（例: PC001） |
| B | 名称 | `name` | 備品の名前 |
| C | ステータス | `status` | 現在の状態（使用可能、修理中など） |
| D | 部署 | `department` | 管理上の部署 |
| E | 購入日 | `purchaseDate` | 備品の購入・導入日 |
| F | 最終点検日 | `lastInspection` | 最後に報告（点検）された日 |
| G | 次回点検日 | `nextInspection` | 次の点検予定日 |
| H | マニュアルURL | `manualUrl` | PDF等のリンク |
| I | 使用場所 | `location` | 【新規】具体的な設置場所 |
| J | 使用部署 | `current_dept` | 【新規】現在使用している部署 |
| K | 備考 | `remarks` | 【新規】自由記述欄 |
| L | 最終所在確認日 | `lastCheck` | 【提案】棚卸し用の自動更新日 |

## 履歴データ シート (Sheet Name: 履歴データ)

*※こちらは基本固定ですが、必要に応じて変更してください*

| 列番号 | フィールド名 | 説明 |
| :--- | :--- | :--- |
| A | 日付 | 報告された日時 |
| B | 備品ID | 対象のID |
| C | 内容 | 定期点検、故障報告など |
| D | 担当者 | 報告した人の名前 |
