# 備品カルテシステム 利用マニュアル（ユーザー向け）

本システムは、ログイン不要（または指定のアカウント連携）で、スマートフォンやPCのブラウザから直接、院内の備品や設備の情報を確認・更新できるWebアプリケーションです。

## 1. システムへのアクセス方法
- **QRコード経由:**
  各備品に貼付されているQRコードをスマートフォンのカメラ等で読み取ってください。
  専用の「備品カルテ（詳細情報）」画面が直接表示され、その備品の情報やマニュアルを即座に確認できます。
- **共有用URL経由:**
  システム管理者から共有されたURL（ブックマーク推奨）にアクセスしてください。
  全備品の一覧画面（インクリメンタル検索機能付き）が開きます。

## 2. 備品を検索・確認する (一覧画面)
一覧画面では、登録されている備品がカード形式でリスト表示されます。
- **キーワード検索:**
  画面上部の検索窓の「備品を検索」に、名称や管理番号、フロアの一部などを入力すると、結果がリアルタイムで絞り込まれます。
- **詳細情報の確認:**
  確認したい備品カードの画像をタップ、またはカード全体をタップすると、その備品の詳細フォーム（単票画面）に切り替わります。
- **取扱説明書の確認:**
  「説明書URL」の項目に **[開く]** リンクがある場合は、タップすることでGoogleドライブ等に保存されたPDFマニュアルを参照できます（アクセス権限が必要です）。

## 3. 備品情報を更新・編集する (詳細画面)
情報を修正する場合は、詳細画面でそのまま項目を書き換えてください。
1. 内容を修正します（例: 「設置場所」を直接書き換える、「状態」を選択するなど）。
2. **備品画像 (任意):** ファイル選択ボタンから写真をアップロードできます。現場で撮影した写真をそのまま登録可能です。
3. 作業を行った方の名前を「**入力担当者名**」欄に入力します。
4. 画面下部の緑色の **[台帳を更新する]** ボタンを押します。
5. 「保存しました」というメッセージが出れば、スプレッドシートへのデータ上書きが完了です。

## 4. メンテナンス履歴を追加する (点検・移動・修理などの報告)
定期点検を実施した際や、備品の場所を移動させた際、あるいは修理に出した際は、**「メンテナンス履歴」** として追記入力を行ってください。
1. 詳細画面の最下部にある、緑色の **「📝 点検・トラブル報告」** ボタンを押します。
2. 報告フォームが開くので、**「報告種別」**（点検、移動、修理、廃棄など）を選択します。
3. **現在の状態** （稼働中、故障中など）を選択します。
4. ※「移動」を選んだ場合は、**「新しい設置階・設置場所」** の入力欄が表示されるため、新住所を入力します。
5. **添付画像 (任意):** 現場の状況や不具合箇所を写真で撮影して添付できます。
6. **報告者名:** 作業を行った方の名前を入力します。
7. 作業内容や故障の状況などを **「報告メモ」** に入力します。
8. **[報告を記録]** ボタンを押すと履歴が追加され、台帳上のステータスやお住まいの場所も自動的に最新情報に書き換わります。

### 過去の履歴を詳しく見る
詳細画面の中ほどにある「メンテナンス履歴」のリスト（タイムライン）の項目をタップすると、**報告当時の詳細内容や拡大写真** をポップアップ画面で確認できます。

## 5. 新しい備品を登録する
新規で購入・導入した備品をシステムに登録する手順です。
1. 共有URLから一覧画面を開き、右下にある青色の **[+ 追加]** ボタン（またはヘッダーの新規登録ボタン）を押します。
2. 空の専用フォームが開きます。
3. **[大分類] → [中分類]** の順にカテゴリを選択します。
4. 備品の名称、型番、設置フロア（プルダウン）、場所などを入力します。
5. **備品画像 (任意):** 備品の全体写真などを登録できます。
6. 必須項目である「入力担当者名」を埋めます。
7. 画面下部の青色の **[台帳に追加する (新規)]** ボタンを押します。
8. 自動的に「管理番号（例: PC2403-01）」が計算され、スプレッドシート（データベース）の末尾に新しい備品データが登録されます。
